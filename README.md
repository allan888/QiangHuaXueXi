# 课程汇报要求

## 课程与汇报安排

- **课程形式**：以小组讨论为主，共3次课，6课时。
- **组队要求**：10人一个小组，周四前完成组队（共6组），名单反馈给课代表。
- **汇报形式**：每个方向2个小组进行PK。
- **时间安排**：35分钟汇报，10分钟质疑（Q&A）。
- **汇报核心**：理论讲解 + 论文分享 + 代码复现。

### 选题方向

1. 强化学习与任务决策（规划、群决策等）
2. 强化学习与路径规划
3. 强化学习与自动控制

### 问题陈述

- 明确任务：输入/输出、目标性能（如 V(s)、期望回报等）
- 指标：如期望回报、收敛速度、方差

### 相关工作/简短对比

- 列出2-3种方法（如 DP、MC、TD）
- 制作一张**对比表格**（包含优缺点），并强调你们选定该方法的理由

### 具体方法/算法

- **总体思路**：使用图示说明 MDP、策略、采样流程
- **关键公式与伪代码**：例如蒙特卡洛评估、重要性采样公式等
  - 要求：仅放核心公式，并逐项讲解
- **算法细节**：算法流程图 / 算法复杂度 / 超参数
- **实验设置**：
  - 环境说明（例如：Taxi-v3 / 3×3 公主营救）
  - 策略 π 与 b、折扣因子 γ、采样数、评价指标、实现语言

### 结果展示

- **主要数值/表格**：平均回报、方差、收敛图
- **关键图表**：收敛曲线、重要性权重分布、误差条
