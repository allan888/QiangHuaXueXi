# 非Q-learning路径规划算法汇总

从 `2006.14195` 和 `Path Planning Method of Mobile Robot Based on Q-learning` 两篇论文中提取的非Q-learning路径规划算法。

---

## 一、来源：2006.14195 综述论文

**论文**: Mobile Robot Path Planning in Dynamic Environments: A Survey  
**作者**: Kuanqi Cai, Chaoqun Wang, Jiyu Cheng, Shuang Song, Clarence W. de Silva, Max Q.-H. Meng

该综述将机器人导航框架分为**全局路径规划**和**局部路径规划**两层，并单独讨论了强化学习方法。以下为非Q-learning算法。

---

### 1.1 全局路径规划算法

全局规划器基于静态全局地图，生成从起点到目标点的无碰撞路径，不考虑动态物体。

#### 1.1.1 图搜索类 (Graph Search)

| 算法 | 原理 | 特点 |
|------|------|------|
| **Dijkstra** | 贪心搜索最短路径，从起点向外层层扩展 | 保证最优解，但计算量大，不适合高维环境 |
| **A\*** | 启发式搜索，结合实际代价 g(n) 和启发式估计 h(n) | 比 Dijkstra 高效，ROS 中广泛实现，但大规模环境计算负担重 |
| **DFS** | 深度优先遍历图/栅格 | 可能找到非最短路径 |
| **BFS** | 广度优先遍历图/栅格 | 保证无权图最短路径，但空间消耗大 |

#### 1.1.2 随机采样类 (Random Sampling)

| 算法 | 原理 | 特点 |
|------|------|------|
| **RRT** (快速探索随机树) | 从起点随机采样扩展树结构，快速探索空间 | 效率高，适用于高维/动态环境；路径非最优 |
| **BIT\*** (批量 informed 树) | 启发式引导的隐式随机几何图搜索，批量采样优化 | 采样效率高，渐进最优 |
| **RABIT\*** (区域加速 BIT\*) | 在 BIT\* 基础上融入局部信息加速 | 进一步提升了规划效率 |
| **Risk-DTRRT** (基于风险的双树 RRT) | 双树 RRT 结合风险评估 | 考虑运动安全性 |

#### 1.1.3 智能仿生类 (Bionic Intelligent)

| 算法 | 原理 | 特点 |
|------|------|------|
| **GA** (遗传算法) | 模拟自然选择和遗传进化，种群迭代优化 | 全局搜索能力强，但收敛慢 |
| **ACO** (蚁群算法) | 模拟蚂蚁信息素寻路行为 | 适合离散优化，易陷入局部最优 |
| **ABC** (人工蜂群算法) | 模拟蜜蜂采蜜的群体智能 | 参数少，鲁棒性强 |
| **PSO** (粒子群优化) | 粒子在解空间中飞行搜索最优解 | 收敛快，易早熟 |
| **OGA-PSO** (优化 GA-PSO 混合) | 遗传算法 + 粒子群混合优化 | 解决了焊接机器人的最短无碰撞路径问题 |
| **改进 ACO** (ACO + 人工势场 + 几何优化) | 蚁群算法结合人工势场和局部几何优化 | 增强全局搜索能力 |

### 1.2 局部路径规划算法

局部规划器利用传感器实时信息对全局路径分段优化，处理动态障碍物和机器人运动约束。

#### 1.2.1 经典局部规划方法

| 算法 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **人工势场法** (APF) | 目标点产生引力、障碍物产生斥力，合力驱动 | 效率高，实时性好 | 存在陷阱区域/局部最小值，通过窄通道时振荡 |
| **模糊逻辑算法** | 基于模糊规则推理导航决策 | 鲁棒性好，对环境信息依赖小 | 规则依赖人工经验，无法学习，灵活性差 |
| **模拟退火算法** | 模拟物理退火过程，以一定概率接受劣解跳出局部最优 | 描述简单，使用灵活，初始条件少 | 收敛速度慢，随机性高 |
| **粒子算法** | 基于群体智能的概率搜索 | 全局优化能力 | 实时性受群体规模影响 |

#### 1.2.2 速度障碍法系列 (Velocity Obstacle, VO)

这是局部规划研究的重点方向，通过定义速度空间中的碰撞区域来实现避障。

| 算法 | 全称 | 核心思想 | 解决的问题 |
|------|------|----------|------------|
| **VO** | Velocity Obstacle | 定义速度空间中会导致碰撞的"锥形"区域，选择锥外的速度 | 开山之作，考虑了障碍物速度 |
| **RVO** | Reciprocal Velocity Obstacle | 两个智能体各自承担一半的避障责任（取当前速度与对方 VO 外速度的平均） | 解决了 VO 在两个智能体迎面时的振荡问题 |
| **HRVO** | Hybrid Reciprocal Velocity Obstacle | 混合互惠策略，防止双方选择同侧通过 | 解决了 RVO 的"互惠舞蹈"僵局 |
| **ORCA** | Optimal Reciprocal Collision Avoidance | 为每个智能体计算无碰撞速度的半平面，线性规划求最优速度 | 多智能体导航的标准化方法 |
| **ClearPath** | — | 选择两 VO 边界线交点或投影最近速度到无碰撞区 | 比 ORCA 更鲁棒 |
| **NH-ORCA** | Non-Holonomic ORCA | ORCA 扩展到非完整约束机器人 | 适用于差速/阿克曼转向机器人 |
| **CALU** | Collision Avoidance with Localization Uncertainty | 结合 ORCA 和 NH-ORCA，减少对先验环境知识的需求 | 处理定位不确定性 |
| **COCALU** | Collision Avoidance under Bounded Localization Uncertainty | 使用粒子云而非外接圆表征机器人 | 处理机器人非圆形底盘 + 定位漂移 |

#### 1.2.3 其他局部规划方法

| 算法 | 原理 | 特点 |
|------|------|------|
| **X-Y-T 空间法** | 在 X-Y-T 三维空间规划，考虑人类朝向和个人空间 | 考虑了人类社交因素；效率取决于网格粒度 |
| **定时 A\*** | A\* 扩展，使用社会成本函数预测人类轨迹 | 可预测行人轨迹，但未考虑障碍物运动 |
| **分层社会成本地图** | 在分层地图上叠加社会成本约束 | 适用于复杂社交环境导航 |

### 1.3 强化学习方法（非Q-learning）

论文第3节专门讨论了 RL 在路径规划中的应用，其中**非 Q-learning** 的方法包括：

| 方法 | 全称/原理 | 优点 | 缺点 |
|------|-----------|------|------|
| **IRL** (逆强化学习) | 从专家演示中反推奖励函数，用于建模人类协作导航 | 可建模人类舒适度模型，提供协作导航 | 计算昂贵，高度依赖特征选择 |
| **CADRL** (碰撞避免 DRL) | 基于深度强化学习的分散式多智能体碰撞避免，在线计算转移至离线训练 | 实时性好，路径质量高；优于 ORCA | 可能导致振荡（边际稳定）路径 |
| **SA-CADRL** | 在 CADRL 中引入时间有效策略处理人类行为随机性 | 可应对人类行为随机性，低速场景下有效 | 未考虑与行人的社会规范关系 |
| **CRL** (复合强化学习) | 通过传感器输入学习社交导航，复合多个学习目标 | 能同时解决社交感知运动规划和人类交互 | 需要先验知识，限制了广泛部署 |
| **LSTM + SA-CADRL** (Everett 扩展) | SA-CADRL 引入 LSTM 预测行人方向 | 不需要假设特定行为模型 | 训练复杂 |
| **阶段训练框架** (Long et al.) | 多种环境和阶段式训练策略 | 可泛化至训练中未出现的新场景 | — |

此外，文中还提到了经典的 **SARSA**（on-policy TD 学习）、**R-learning**（average-reward RL）、**Q(λ)**（结合资格迹加速收敛），但这些与 Q-learning 有较强关联。

## 二、来源：Path Planning Method of Mobile Robot Based on Q-learning

**论文**: Path Planning Method of Mobile Robot Based on Q-learning  
**作者**: Qi Jiang（西南交通大学），2022

该论文本身采用 Q-learning 方法，但其**引言**中列举了以下非 Q-learning 的路径规划算法作为相关工作：

| 算法 | 类型 |
|------|------|
| **遗传算法 (GA)** | 智能仿生 / 进化计算 |
| **粒子群优化 (PSO)** | 群体智能 |
| **人工势场法 (APF)** | 经典局部规划 |
| **A\* 算法** | 启发式图搜索 |
| **Dijkstra 算法** | 图搜索 |
| **神经网络方法** | 深度学习 |
| **改进协同进化遗传算法** | 多机器人全局路径规划（Qu et al., 2013） |

## 三、算法分类总览

```
非Q-learning路径规划算法
├── 全局路径规划 (Global)
│   ├── 图搜索: Dijkstra, A*, DFS, BFS
│   ├── 随机采样: RRT, BIT*, RABIT*, Risk-DTRRT
│   └── 智能仿生: GA, ACO, ABC, PSO, OGA-PSO
├── 局部路径规划 (Local)
│   ├── 经典方法: 人工势场法, 模糊逻辑, 模拟退火, 粒子算法
│   ├── 速度障碍系列: VO → RVO → HRVO → ORCA → ClearPath
│   │                  ├── NH-ORCA (非完整约束)
│   │                  ├── CALU (定位不确定性)
│   │                  └── COCALU (有界不确定性 + 非圆形底盘)
│   └── 时空方法: X-Y-T空间法, 定时A*, 分层社会成本地图
└── 强化学习 (非Q-learning)
    ├── IRL (逆强化学习)
    ├── CADRL (碰撞避免DRL)
    ├── SA-CADRL + LSTM
    ├── CRL (复合强化学习)
    └── 阶段训练框架
```

## 四、关键对比：Q-learning vs 非Q-learning

| 维度 | Q-learning 类方法 | 非Q-learning 方法 |
|------|-------------------|-------------------|
| **代表算法** | Q-learning, DQN, Double DQN, Dueling DQN | A*, Dijkstra, RRT, VO, APF, GA, PSO, IRL |
| **是否需要模型** | 无模型 (model-free) | 图搜索/采样需环境模型；VO 需速度信息；仿生算法无模型 |
| **处理动态障碍** | 通过在线学习适应动态环境 | VO 系列原生支持动态；A*/Dijkstra 仅静态 |
| **实时性** | 训练慢，推理快 | A*/Dijkstra 极快但仅静态；VO 系列实时性好 |
| **最优性保证** | 无理论保证（收敛到最优策略需无限探索） | A*/Dijkstra 保证最优；RRT 非最优（渐进最优）；VO 局部最优 |
| **可解释性** | 低（Q表 / 神经网络黑箱） | 高（明确数学原理） |
| **泛化能力** | 需在新环境重新训练 | 图搜索/采样无需训练；VO 依赖速度建模 |
