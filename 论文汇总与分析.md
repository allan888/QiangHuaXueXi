# 论文汇总与分析

本目录下共有7篇论文，按照研究内容分为四类：**综述、静态寻路、动态寻路、优化代码（算法改进）**。

---

## 一、综述类

### 1. Mobile Robot Path Planning in Dynamic Environments: A Survey
- **文件**: `2006.14195.pdf`
- **作者**: Kuanqi Cai, Chaoqun Wang, Jiyu Cheng, Shuang Song, Clarence W. de Silva, Max Q.-H. Meng（哈工大深圳 / 香港中文大学 / UBC）
- **核心逻辑**:
  - 将机器人导航框架分为**全局路径规划**和**局部路径规划**两个层次
  - 全局规划方法三类：图搜索（Dijkstra、A*）、随机采样（RRT、BIT）、智能仿生（GA、ACO、PSO）
  - 局部规划重点分析**Velocity Obstacle (VO)** 及其变体（RVO、HRVO、ORCA、ClearPath、CALU、NH-ORCA、COCALU），这些方法通过定义速度空间中的碰撞区域来实现避障
  - 详细介绍了**强化学习在路径规划中的应用**：Q-learning、IRL（逆强化学习）、CADRL（碰撞避免DRL）、SA-CADRL、CRL（复合强化学习）
  - 结论：RL方法在长期来看可能被编码进分层路径规划框架中

## 二、静态寻路

### 2. Path Planning Method of Mobile Robot Based on Q-learning
- **文件**: `Path_Planning_Method_of_Mobile_Robot_Based_on_Q-le.pdf`
- **作者**: Qi Jiang（西南交通大学）
- **发表**: Journal of Physics: Conference Series, 2022
- **类别**: 静态寻路
- **核心逻辑**:
  - 使用栅格法在MATLAB中建立20×20的静态障碍物环境
  - 动作空间：9个动作（上、下、左、右、左上、左下、右上、右下、静止）
  - 奖励函数：到达终点+1，碰到障碍物-0.3，其他情况0（为鼓励探索）
  - 参数设置：学习率α=0.6，折扣率γ=0.9，迭代80次
  - 改进了原始Q-learning的低效率和慢收敛问题
  - 与遗传算法和A*算法进行了对比，Q-learning在陌生环境中具有独立规划和适中规划时间的优势

## 三、动态寻路

### 3. Path Planning in Dynamic Environments Based on Q-Learning
- **文件**: `ICMSC2023-222-230.pdf`
- **作者**: Xiangqi Li（中国海洋大学）
- **发表**: ICMSC 2023
- **类别**: 动态寻路
- **核心逻辑**:
  - 在MATLAB中创建三个不同复杂度的动态环境（10×10网格）
    - 环境1：1个静态障碍物 + 1个动态障碍物
    - 环境2：2个静态障碍物 + 1个动态障碍物
    - 环境3：多个静态障碍物 + 2个动态障碍物
  - 动态障碍物以与机器人相同的速度移动
  - 引入"垃圾清理车"任务模型：机器人必须先到达**垃圾点（trash point）**再到达终点
  - 动作空间：环境1/2用4个动作（上下左右），环境3用8个动作（增加对角线）
  - 奖励设计：撞障碍物/边界=-1，到达垃圾点=+5，到达终点=+5（环境3中垃圾点奖励提高到+10）
  - 学习率α固定0.25，折扣率γ从0.99开始每100轮减0.01
  - 每个环境都能生成3条不同的最优路径，Q值在起点(1,1)处正确反映了最优动作方向
  - **发现的问题**：Q-learning缺乏全局规划能力，会产生冗余动作（如先右后上而非直接右上）

### 4. Q-learning-based Unmanned Aerial Vehicle Path Planning with Dynamic Obstacle Avoidance
- **文件**: `main-2.pdf`
- **作者**: Amala Sonny, Sreenivasa Reddy Yeduri, Linga Reddy Cenkeramaddi（挪威阿格德尔大学）
- **发表**: Applied Soft Computing, 2023
- **类别**: 动态寻路（含算法优化）
- **核心逻辑**:
  - 针对**无人机（UAV）**在包含静态和动态障碍物环境中的路径规划
  - 提出**Shortest Distance Prioritization (SDP) 策略**：在每个状态下，不仅选择Q值最高的动作，还检查该动作是否使agent更接近目标点，给予更近的动作更大权重
  - 25×25网格环境，4个静态障碍物，最多4个动态障碍物
  - 设计了4个版本的算法对比：
    - 原始Q-learning（仅静态障碍物）
    - 原始Q-learning（静态+动态障碍物）
    - 提出的Q-learning+SDP（仅静态障碍物）
    - 提出的Q-learning+SDP（静态+动态障碍物）
  - 与A*、Dijkstra、SARSA对比：A*和Dijkstra虽然更快（~0.01s）但无法处理动态障碍物；SARSA速度慢且路径长；原始Q-learning在动态环境中需要约10倍训练时间
  - 提出的SDP策略大幅缩短了训练时间和路径长度

### 5. 基于强化学习的机器人路径规划算法（Robot Path Planning Algorithm Based on Reinforcement Learning）
- **文件**: `基于强化学习的机器人路径规划算法_张福海.pdf`
- **作者**: 张福海, 李宁, 袁儒鹏, 付宜利（哈尔滨工业大学 机器人技术与系统国家重点实验室）
- **发表**: 华中科技大学学报（自然科学版）, 2018
- **类别**: 动态寻路
- **核心逻辑**:
  - 利用**激光雷达（LIDAR）**获取的周围障碍物信息离散化为有限状态
  - 状态空间设计（共1792个状态）：
    - Rg：目标点相对机器人的方位角，离散为7个扇区
    - Ro1~Ro4：前方120°范围内障碍物按角度和距离分为4个区域，每个区域4个离散级别
  - 动作设计：考虑机器人运动学约束，分为前进(0.3, 0.0)、左转(0.1, -0.6)、右转(0.1, 0.6)，并设置动态窗口保证运动连续性
  - **连续报酬函数**的设计是核心创新：每个动作都能获得即时反馈
    - 到达目标点（<阈值d_win）→正报酬r_rea
    - 碰撞障碍物（<阈值d_col）→负报酬r_col
    - 其他情况→报酬 = 机器人距目标点距离的变化值（靠近为正，远离为负）
  - Gazebo仿真：10m×7m场景，5000次训练，30小时，成功率收敛至95%
  - **实物实验**：在5.4m×8.0m真实环境中验证，三次导航均成功（路径长度约4.7-5.0m，时间约30s）

## 四、优化代码（算法改进）

### 6. 基于强化学习的移动机器人路径规划优化
- **文件**: `基于强化学习的移动机器人路径规划优化.pdf`
- **作者**: 尹旷, 王红斌, 方健, 莫文雄, 叶建斌, 张宇（广东电网广州供电局电力试验研究院）
- **发表**: 电子测量技术, 2021
- **类别**: 代码优化（改进Q-learning收敛速度）
- **核心逻辑**:
  - 针对Q-learning中**探索（exploration）与利用（exploitation）的平衡问题**
  - 提出**ε-decreasing动作选择策略**：ε因子不再固定，而是根据智能体对环境信息的掌握程度动态下降
    - 公式：ε = e^(-(Q(S_t, a_b) - Q(S_t, a_t)) / n)，其中n = M - m（M为最大迭代次数，m为当前迭代次数）
    - 初始时智能体对环境一无所知，ε≈1（纯探索）
    - 随着迭代深入，ε逐渐减小（转为利用学到的知识）
  - **动态学习率**：探索阶段用较大学习率α=0.7，利用阶段用较小学习率α=0.1
  - 仿真环境：20×20栅格地图，MATLAB
  - 结果：与基础Q-learning相比，在获得相同最优路径（长度36）的前提下，**迭代次数减少约32%**（从591次降至402次）
  - 后期收敛曲线更平缓，避免了基础Q-learning收敛后期仍然剧烈震荡的问题

### 7. 改进奖励函数的深度强化学习路径规划方法
- **文件**: `改进奖励函数的深度强化学习路径规划方法.pdf`
- **作者**: 桑垚, 马晓宁（中国民航大学）
- **发表**: 计算机应用与软件, 2025
- **类别**: 代码优化（改进深度强化学习的奖励函数）
- **核心逻辑**:
  - 针对**深度强化学习中的稀疏奖励问题**（传统方法仅在回合结束时给予奖励）
  - 提出**PB-DuelingDQN（基于潜能的双竞争Q网络）**模型
  - 奖励函数改进（基于潜能设计）：
    - 原始奖励只能从终点/碰撞获得（稀疏），导致样本利用率低、训练困难
    - 新奖励函数加入即时奖励：F(s,a,s') = γΦ(s') - Φ(s)，其中Φ(s)为智能体到目标点的欧氏距离
    - 即每一步都给予即时奖励 = 当前距离 - 下一步距离（靠近得正奖励，远离得负奖励）
  - 网络架构：使用Dueling DQN（将Q值分解为状态价值V(s)和动作优势A(s,a)）+ 双网络（估计网络+目标网络）避免过估计
  - 输入：栅格地图图像 → 三层卷积+池化 → 输出4个动作的Q值
  - 实验设计了三组对比：
    - 实验一（验证新奖励函数）：PB-DuelingDQN vs DuelingDQN
    - 实验二（验证有效性）：PB-DuelingDQN vs DDQN vs DQN
    - 实验三（验证先进性）：PB-DuelingDQN vs SAC-Discrete
  - 结果（简单地图16×16 / 复杂地图28×28）：
    - 寻路成功率：87.01% / 62.92%（比DuelingDQN分别提高7.08和12.60个百分点）
    - 平均成功寻路长度：14.94 / 24.22（比DuelingDQN更短）
    - 与SAC-Discrete相比成功率接近，但路径长度明显更短

## 论文之间的关联与技术脉络

1. **从静态到动态**：Qi Jiang（论文2）的静态Q-learning → Xiangqi Li（论文3）加入动态障碍物 → Sonny（论文4）进一步处理UAV场景的多动态障碍物

2. **从传统RL到深度RL**：张福海（论文5）的手工状态离散化（1792个状态）→ 桑垚（论文7）用CNN自动提取特征，解决维数灾难

3. **奖励函数设计的演进**：
   - 基础：到达终点+1，碰撞-1（稀疏）
   - 张福海（论文5）：连续报酬函数，每一步都有距离变化反馈
   - 桑垚（论文7）：基于潜能的即时奖励，理论上保证最优策略不变

4. **探索-利用平衡的改进**：
   - 基础：固定ε-greedy
   - 尹旷（论文6）：ε随环境熟悉度自适应下降
   - 桑垚（论文7）：ε从0.9999975指数衰减

5. **从仿真到实物**：张福海（论文5）是唯一进行了实际机器人导航实验验证的论文
