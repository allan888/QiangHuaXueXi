# 基于动态快速Q-learning的无人水面艇路径规划方法

**原文标题**: A path planning approach for unmanned surface vehicles based on dynamic and fast Q-learning  
**作者**: Bing Hao, He Du, Zheping Yan  
**期刊**: Ocean Engineering 270 (2023) 113632  
**DOI**: https://doi.org/10.1016/j.oceaneng.2023.113632

---

## 摘要

路径规划是无人水面艇（USV）的关键问题，有效的路径规划算法能够使USV完成其任务。本文提出了一种名为动态快速Q-learning（DFQL）的新型算法，用于解决部分已知海洋环境中USV的路径规划问题。该算法将Q-learning与人工势场法（APF）相结合来初始化Q表，为USV提供来自环境的先验知识。为了加速Q-learning收敛到最优解，并避免USV在探索初期随机游走的行为，本文提出了静态和动态奖励来激励USV向目标移动。此外，通过在不同环境条件下的离线和在线两种模式验证了所提算法的性能。与现有方法的比较表明，所提方法对USV的路径规划是有效的。

**关键词**: 无人水面艇、路径规划、Q-learning、离线、在线

## 1. 引言

近年来，随着全球对海洋资源关注度的提高，以及材料科学、机械工程、控制理论、传感器、人工智能等技术的快速发展，无人水面艇（USV）应运而生。USV从平台上移除了人为干预，本质上表现出高度非线性动力学特性。USV因其可靠、快速、高度机动等特点，被广泛应用于科学研究、海洋资源勘探、军事用途、水上救援、环境任务等领域。

路径规划在USV成功完成任务中起着至关重要的作用，因为海洋环境具有复杂多变的特点。路径规划是在地图或网格中，计算从起始位置到目标位置的可行路径，且途中不与障碍物发生碰撞。

目前已有多种算法可以实现USV的路径规划：

- **A\*算法**：最流行的基于搜索的路径规划算法之一，从起点开始检查相邻方格，向外扩展直到找到目标。搜索类算法易于应用于路径规划任务，但计算时间随地图大小呈指数增长。
- **快速探索随机树（RRT）**：一种基于采样的路径规划算法，通过状态空间中采样点的碰撞检测，避免了空间建模，能有效解决高维空间和复杂约束的路径规划任务。然而，由于采样点是随机生成的，得到的路径不够平滑且长度过长。
- **智能优化算法**：如粒子群优化（PSO）、遗传算法（GA）等受自然启发或模仿人类思维建模的算法。这类算法具有全局优化性能强、通用性好、适合并行处理等优点，但容易陷入局部最小值，无法得到最优路径。

上述算法各有优缺点，但大多数不能同时适用于离线和在线路径规划，且路径长度和角度不够理想。当忽略USV尺寸将其视为质点时，搜索得到的路径过于靠近障碍物，容易发生碰撞。

为克服上述局限性，本文采用一种基于强化学习的改进Q-learning算法，来解决部分已知环境下USV的离线和在线路径规划问题，并将该算法称为动态快速Q-learning（DFQL）。所提算法充分利用了Q-learning在路径规划中的优势，最小化其迭代次数和计算时间，使USV能够在离线和在线两种模式下以安全距离避开复杂海洋环境中的障碍物。该算法还能使USV从被障碍物阻挡的死胡同路径中逃脱。

### 1.1 相关工作

USV的路径规划、碰撞避免和能效问题近年来已成为热门话题，越来越多的学者和研究人员提出了新颖的想法和创新算法来解决这些问题。

- **Yu等人（2021）** 改进了A\*和APF算法来解决USV的全局和局部路径规划问题，但路径长度不是最优的，且未考虑海洋环境中的动态障碍物。
- **Yao等人（2021）** 开发了一种基于D\* lite算法的新型路径规划方法，适用于复杂环境下USV的实时路径规划，但路径长度和转弯角度不是最优的。
- **Liu等人（2019）** 将PSO与对立学习（OBL）相结合来解决USV的全局和局部路径规划问题，但计算的路径长度和平滑度不够理想。
- **Sang等人（2021）** 提出了一种称为多子目标人工势场法（MTAPF）的新型确定性算法，但该算法未考虑海洋环境中的动态障碍物。

据我们所知，之前的工作没有专门针对USV同时进行离线和在线模式路径规划的问题。因此，本文提出了DFQL算法来解决这一问题。

### 1.2 原创贡献

本文提出的DFQL算法应用于解决USV的离线和在线路径规划问题，主要贡献如下：

1. 针对USV受波浪干扰无法沿既定路径行驶的问题，提出了安全距离以避免与障碍物碰撞；
2. 将Q-learning与APF相结合初始化Q表，避免算法初期在起始位置附近随机移动、收敛速度慢和计算时间长的问题；将常数奖励改为静态和动态奖励值组合，加速算法收敛；改进算法在复杂障碍物环境下容易陷入死胡同、无法到达目标位置的缺点；
3. 所提DFQL算法可为USV在海洋环境中进行离线和在线路径规划，避开静态和动态障碍物；
4. 通过与其他算法比较，证明DFQL能在考虑路径长度、碰撞避免和平滑性的前提下完成路径规划任务。

## 2. 预备知识

### 2.1 问题描述

由于海面工作环境中存在波浪干扰，本文定义了USV安全距离半径rsd，以避免规划路径与障碍物发生碰撞。仿真环境按照USV安全距离，在笛卡尔坐标系中划分为边长为2rsd的7x7网格地图。白色网格γ代表可行区域；黑色网格Sobs代表障碍物（陆地、船只和其他海面障碍物）。

环境中无障碍物时，USV可沿8个相邻方向移动（上、右上、右、右下、下、左下、左、左上）；当位于环境边缘时，可移动方向均指向环境内部。

### 2.2 Q-learning算法

强化学习是一类无监督算法，强调智能体与环境交互的学习过程。智能体的目标是通过学习状态-动作价值函数来评估每个状态下应选择的动作。智能体不被告知应采取何种动作，而必须在每个状态下选择使累积奖励值最大化的最优动作。USV通过持续反馈与环境交互，产生更多数据（状态和回报），并利用新数据进一步改善其行为。

Q表是智能体在状态st下采取动作at所能获得的期望收益，更新方程如式(1)：

```
Q(st, at) ← (1-α) * Q(st, at) + α * [rt+1 + γ * max_a Q(st+1, at+1)]   (1)
```

等价地：
```
Q(st, at) ← (1-α) * Q(st, at) + α * [rt+1 + γ * V(st+1)]                (2)
V(st) ← V(st) + α * [rt+1 + γ * V(st+1) - V(st)]                        (3)
V*(st) = max_a Q(st, at)                                                 (4)
```

其中 α（0≼α≼1）为学习率参数，γ（0≼γ≼1）为折扣率参数。

本文选择Q-learning算法应用于USV路径规划的原因：
- 强化学习与环境具有优越的交互性，无需正负标签。USV通过探索和学习海洋环境获取当前知识，改善运行策略以适应环境。
- Q-learning参数少、易于实现，采用贪心策略。
- Q-learning采用off-policy策略，动作选择依据目标策略，不仅适用于USV离线规划，也适用于在线路径规划。

## 3. 所提出的DFQL算法

本文讨论的USV路径规划问题有以下前提：
- 实际中USV和障碍物环境是三维物体，但本文忽略高度，将其视为二维物体。
- 具备准确导航和控制机器人移动到指定位置的能力，忽略规划过程中传感器测量不确定性或动力学对USV的影响。
- USV路径规划分为离线和在线两种模式：离线模式下，环境中障碍物的位置、形状和尺寸对USV完全已知；在线模式下，环境部分已知或完全未知，USV配备必要传感器在航行中识别障碍物。

DFQL算法分为五个阶段来解决USV离线和在线路径规划问题。

### 3.1 Q表初始化

为优化Q-learning收敛速度慢和计算时间长的问题，本文采用人工势场法与Q-learning算法相结合来优化初始Q表。原因是：人工势场法可在离线和在线两种模式下与Q-learning算法结合，易于在网格地图中实现，能为USV提供环境先验知识，加快计算和收敛速度。

#### 3.1.1 离线路径规划的Q表初始化

离线模式下，环境信息（起始位置、目标位置、障碍物形状、尺寸和位置坐标）对USV完全已知，因此使用库仑定律对网格地图进行APF建模。

引力场：
```
Ua1(st) = (1/2) * ka * ρg²(st)                                      (5)
```

斥力场：
```
Ur1(st) = (1/2) * kr * [1/ρob(st) - 1/ρo]², 当 ρob(st) ≼ ρo         (6)
Ur1(st) = 0,                                当 ρob(st) > ρo
```

总势能：
```
Un1(st) = Ur1(st) + Ua1(st)                                          (7)
```

归一化势能：
```
U1(st) = |Umax1 - Un1(st)| / Umax1                                   (8)
```

初始Q表：
```
Q01(s0, a0) = U1(st)                                                 (9)
```

其中 Ua1(st) 为状态st处产生引力的引力场；Ur1(st) 为产生斥力的斥力场；Un1(st) 为状态st的总势能；ρg(st) 为状态st与目标位置中心的欧几里得距离；ρob(st) 为状态st与障碍物中心的欧几里得距离；ρo 为障碍物影响因子；ka 和 kr 为比例因子；Umax1 为状态st中的最高势能。

#### 3.1.2 在线路径规划的Q表初始化

在线模式下，环境信息对USV部分已知或完全未知，USV仅在已知起始位置和目标位置的条件下，通过传感器探测周围障碍物到达目标位置。因此忽略斥力部分，仅计算目标位置产生的引力来优化初始Q表。

引力场：
```
Ua2(st) = (1/2) * ka * ρg²(st)                                      (10)
```

总势能（仅有引力）：
```
Un2(st) = Ua2(st)                                                    (11)
```

初始Q表：
```
Q02(s0, a0) = U2(st) = |Umax2 - Un2(st)| / Umax2                    (12)
```

### 3.2 动作选择

为避开障碍物并以最短路径长度到达目标位置，定义了USV的8个移动方向（上、右上、右、右下、下、左下、左、左上）。根据安全距离定义（rsd=5m），对应的移动距离为：

- a1: 向上移动 10m
- a2: 向右上移动 10√2 m
- a3: 向右移动 10m
- a4: 向右下移动 10√2 m
- a5: 向下移动 10m
- a6: 向左下移动 10√2 m
- a7: 向左移动 10m
- a8: 向左上移动 10√2 m

### 3.3 奖励函数

奖励函数用于判断行为的价值，USV根据奖励函数与环境交互，通过奖励值调整动作策略。正确设置奖励函数有助于强化期望行为并惩罚不当行为。以往的强化学习中，奖励值通常是静态常数，导致智能体在环境中随机搜索，增加了收敛时间。

为减少计算量，本文使用曼哈顿距离进行计算。奖励函数如式(13)-(17)：

```
reward = rs * (1 + rd)                                               (13)
```

静态奖励 rs：
```
rs = 1,  当st+1为水平/垂直移动                                                                                         (14)
rs = √2, 当st+1为对角移动
rs = 10, 当st+1为目标节点
rs = -inf, 当st+1为禁行节点
```

曼哈顿距离：
```
dt = |ytarget - yt| + |xtarget - xt|                                  (15)
dt+1 = |ytarget - yt+1| + |xtarget - xt+1|                            (16)
```

动态奖励：
```
rd = (dt - dt+1) / |dt - dt+1|                                       (17)
```

其中 rs 为静态奖励；rd 为动态奖励；dt 为状态st到目标位置的曼哈顿距离；dt+1 为下一状态st+1到目标位置的曼哈顿距离；(xt, yt) 为状态st的坐标；(xt+1, yt+1) 为状态st+1的坐标；(xtarget, ytarget) 为目标位置的坐标。

根据此公式建立动态奖励值，当USV接近目标位置时奖励值较大，远离目标位置时奖励值较小，避免USV随机搜索过程消耗能量并加速收敛。

### 3.4 DFQL的收敛性

由于Q-learning的更新公式发生了变化，需要重新证明所提算法的收敛性。

设 Q̂n(st, at) 为第n次更新时 Q(st, at) 的估计值，定义误差：

```
δn = |Q̂n(st, at) - Q(st, at)|                                       (18)

δn+1 = |Q̂n+1(st, at) - Q(st, at)|
      ≼ |(1-α)(Q̂n(st, at) - Q(st, at))| + αγ * max_a |Q̂n(st+1, at+1) - Q(st+1, at+1)|
      ≼ (1-α)δn + αγδn = δn                                          (19)
```

随着DFQL算法迭代，Q值收敛到一个确定值。USV将以最优动作策略从起点移动到目标点。

### 3.5 离线和在线路径规划的DFQL

DFQL算法应用于USV离线和在线路径规划的基本思想是：基于Q-learning算法与环境的交互，分别在离线和在线模式下利用APF初始化Q表，为USV提供环境中的先验知识，避免探索初期的随机选择动作，提高环境探索效率，加速收敛和计算；结合静态和动态奖励值优化奖励函数，诱导USV向目标位置移动，提高原始算法的计算效率，输出最短路径。

综上，USV使用DFQL算法离线和在线模式的流程如下：当USV位于起始位置时，对环境中已知障碍物执行离线路径规划；USV按照离线路径行驶，当船载传感器检测到USV附近存在未知障碍物时，切换到在线模式避开未知障碍物；之后切换回离线模式，若检测到USV周围有未知障碍物，重复上述步骤直到USV到达目标位置。

## 4. 仿真结果

### 4.1 评价函数

#### 4.1.1 路径长度

路径长度定义为USV从起始位置到目标位置的欧几里得距离之和。

#### 4.1.2 转弯角度

转弯角度定义为USV从起点到目标点航向角变化的总和。

#### 4.1.3 CPU时间

CPU时间即算法通过迭代计算所消耗的时间（秒）。计算时间越短，USV执行动作的等待时间越少。

### 4.2 环境配置

本文将USV工作的海洋环境简化为长30格、宽20格的矩形（记为M）。仿真地图使用M01至M06表示。以半径rsd=5m的圆表示USV的安全距离，以边长为2rsd=10m的正方形表示工作空间中的每一格，白色网格表示可行区域，黑色网格表示障碍物。

### 4.3 参数选择

在DFQL算法执行前，需要确定Q值更新方程中学习率α和衰减率γ的值。根据Watkins等人的理论，当α值较小时，智能体遍历环境中的所有状态并计算所有可能动作，Q值收敛到最优值；当γ值较大时，可扩大智能体的探索范围，防止陷入局部最优。

本文固定γ=0.9，记录路径长度收敛到最优值所需的迭代次数。重复测试30次取平均值，结果显示α值与收敛迭代次数成反比。综合考虑USV的计算时间，α和γ均取0.9。

### 4.4 USV的离线和在线路径规划

#### 4.4.1 离线路径规划

离线模式下环境中的障碍物对USV完全已知，目标是在起始状态和目标状态之间计算一条可行路径。设计了M01至M06的仿真环境和实验结果，模拟USV在不同工作环境（如群岛、河流和港口）中的情况。

为评估DFQL算法的通用性和普适性，在6种海洋仿真环境中与以下算法进行比较：
- **CQL**（经典Q-learning）
- **PSO**（粒子群优化）
- **GWO**（灰狼优化）
- **DA**（蜻蜓算法）
- **MFO**（飞蛾火焰优化）

每种算法在每个仿真环境中重复30次，记录路径长度、转弯角度和计算时间的均值、标准差及t检验结果。

**离线实验结果**：
1. 在所有情况下，DFQL算法规划的USV全局路径具有最短路径长度和最小转弯角度；
2. 在所有情况下，DFQL算法具有最小的标准差，证明了其在不同环境下的优越性和稳定性；
3. DFQL相比CQL计算时间有改善，但比PSO、GWO、DA和MFO稍长。

与CQL相比，DFQL的改进百分比为：
- 路径长度：减少1.57%–3.13%
- 转弯角度：减少11.11%–55.56%
- 计算时间：减少2.91%–6.88%

**早熟性分析**：DFQL算法30次迭代中未出现早熟现象，能计算出最优解；而PSO、GWO、DA和MFO在障碍物稀疏的环境中更容易出现早熟。

#### 4.4.2 在线路径规划

在实际中环境并非完全已知，存在部分已知或完全未知的情况。在动态和不确定的海洋环境中，路径规划前难以甚至不可能获得各种障碍物的信息。这要求USV配备传感器（雷达、激光雷达、电光相机、红外相机等）来获取周围船舶的位置、航向和速度信息。

为证明DFQL算法在在线模式下的有效性和通用性，与RRT和APF进行比较。

**在线静态障碍物路径规划**：

在离线路径上设置随机放置的静止障碍物（在实际海洋环境中可以是静止的船舶、漂浮物、礁石等）。实验结果表明DFQL在USV在线路径规划方面（路径长度和计算时间）显著优于比较算法，但计算时间较长。

- RRT每次得到的路径长度和转弯角度随机性过大（标准差大），不适合USV路径规划。
- APF虽然每次得到的路径长度和转弯角度稳定性好，但需要调整相应参数以避免陷入局部最小值问题。当障碍物过于密集时，APF无法计算出结果。
- RRT和APF计算的路径过于靠近障碍物，不利于USV规划路径。

**在线动态障碍物路径规划**：

真实海洋环境中的障碍物并非完全静止，在移动船舶和洋流气候作用下存在动态障碍物。本文将动态障碍物的运动简化为匀速直线运动（每次移动一格）。

实验表明DFQL算法在USV在线路径规划中能有效避开动态障碍物，路径长度和计算时间显著优于比较算法。实验结果证明了所提方法的实用性、通用性和鲁棒性。局限性在于相比比较算法计算时间较长，且会随地图大小呈指数增长。

---

## 5. 结论

本文提出了用于USV路径规划的DFQL算法，并根据USV在不同海洋环境中的离线和在线模式进行了仿真实验。离线模式下的环境完全已知，在线模式下的静态和动态障碍物部分已知或完全未知。两种模式的切换可有效解决USV路径规划问题，节省能量和时间。

通过将APF与Q-learning算法相结合来初始化Q表，并设置静态和动态奖励函数为USV提供环境先验知识，加速了CQL的收敛速度和计算时间。

USV实际工作环境中的障碍物并非完全已知。当传感器探测到USV周围的静态和动态障碍物并切换到在线模式使用DFQL算法时，仿真实验和数据分析表明DFQL算法得到的路径和平滑度显著优于其他比较算法，使USV能实现避障。

**未来工作**：将研究多USV路径规划。为减少计算时间和协调多USV，将使用深度强化学习进行研究。
