## 2. USV 和障碍物物理模型（长x宽矩形）

### USV

| 属性 | 值 |
|---|---|
| 长度 | `USV_LENGTH = 25m` |
| 宽度 | `USV_WIDTH = 8m` |
| 航向 | 8个离散方向 (0°, 45°, ..., 315°) |

### 障碍物

| 类型 | 数量 | 尺寸 |
|---|---|---|
| 静态障碍物 | 4个 | 矩形，轴对齐 (O1~O4) |
| 动态障碍物 | 2个 | 30m x 15m，每步随机移动10m |

### 碰撞检测

- 使用 **Separating Axis Theorem (SAT)** 进行 OBB-OBB 重叠检测 (`obb_overlap()`)
- 预计算静态障碍物占用网格 `_static_occ_grid` (100x100 bool)，快速预过滤：
  - 采样点所在格元被占用 → 执行完整OBB检测
  - 采样点所在格元未占用 → 仅检查动态障碍物

## 3. 回转半径 + 圆弧转弯

### 参数

| 参数 | 范围 |
|---|---|
| 最小回转半径 | `USV_MIN_RADIUS = 15m` |
| 最大回转半径 | `USV_MAX_RADIUS = 60m` |

### 转弯机制

- **直行动作 (FORWARD)**: 沿当前航向移动 `CELL_SIZE = 10m`
- **左转 (TURN_LEFT)**: 航向逆时针旋转45°，位置沿圆弧移动
- **右转 (TURN_RIGHT)**: 航向顺时针旋转45°，位置沿圆弧移动

### 半径选择 (`_choose_turn_radius`)

模型在 `[Rmin, Rmax]` 范围内测试 **5个候选半径**（从大到小），选择能避开所有障碍物的最大半径：

```
candidates = [Rmax, ..., Rmin]  # 5个等距值
for R in candidates:
    arc = compute_turn_arc(x, y, heading, turn_action, R)
    if no_collision(arc):
        return R
return None  # 无法转弯
```

优先选大半径 → 转弯更平缓。

### 圆弧计算 (`compute_turn_arc`)

```
给定: 当前位置 (x,y), 航向 θ, 转弯方向, 半径 R
转弯角度: Δθ = 45° = π/4
弧长: s = R * π/4

右转 (CW):
  圆心: (x + R·sinθ, y - R·cosθ)
  弧参数角: 从 θ+π/2 到 θ+π/2-Δθ

左转 (CCW):
  圆心: (x - R·sinθ, y + R·cosθ)
  弧参数角: 从 θ-π/2 到 θ-π/2+Δθ

圆弧离散化为 12 个采样点用于碰撞检测
```

## 6. 运行结果

| 指标 | Alg2 (ε-greedy) | Alg4 (SDP-QL) |
|---|---|---|
| 训练成功次数 | 0/2000 | 197/2000 |
| 路径点数 | 31 | 34 |
| 到达终点 | 否 | 是 |
| 训练耗时 | ~58s | ~442s |

Alg2 的 ε-greedy 在80K状态空间中无法通过随机探索发现稀疏的 +1 目标奖励，属于预期行为。Alg4 的 SDP 启发式策略有效引导USV完成物理约束下的路径规划。
