# 模型输入输出说明

## 输入量

每步输入是一个状态三元组 **`(grid_x, grid_y, heading_idx)`**：

| 分量 | 来源 | 范围 | 说明 |
|------|------|------|------|
| `grid_x` | `pos_to_grid(x, y)` | 0–99 | USV 连续坐标 `(x,y)` 离散化到 100×100 网格（10m/格） |
| `grid_y` | `pos_to_grid(x, y)` | 0–99 | 同上 |
| `heading_idx` | `usv.heading_idx` | 0–7 | 8 个离散航向角（0°, 45°, 90°...315°） |

USV 实际运动在 `(0, 1000)` 连续坐标空间上，Q 表索引用的是离散网格。

## 输出量

输出是 **3 选 1** 的离散动作：

| 动作 | 值 | 连续空间的实际效果 |
|------|----|--------------------|
| `FORWARD` | 0 | 沿当前航向角**直线前进** 10m（一格），等距采样 15 个点做扫掠碰撞检测 |
| `TURN_LEFT` | 1 | **向左画弧**，从 8 个转弯半径中选第一个无碰撞的（60m→15m），沿弧线移动后航向角 +1 |
| `TURN_RIGHT` | 2 | **向右画弧**，同理尝试多个半径，沿弧线移动后航向角 -1 |

转弯半径选择逻辑（第 267-272 行）：

```python
candidates = np.linspace(max_radius=60, min_radius=15, 8)  # 60→15m, 8 个档位
for R in candidates:
    xs, ys = compute_turn_arc(...)         # 弧线离散采样 12 点
    if not _check_swept_collision(...):    # 逐点 OBB 碰撞检测
        return R                           # 选第一个无碰撞的半径
```

- 大半径 = 平缓转弯（50-60m），需要空间大
- 小半径 = 急转弯（15-20m），可在狭窄空间内转向
- 虽然动作只有 3 个，但 8 航向 × 8 转弯半径 = **64 种可能的转弯轨迹**

## Q 表结构

```
q_table[100][100][8][3]  →  240,000 个 Q 值
         ↓     ↓  ↓  ↓
      gridX gridY hdg action
```

状态 × 动作空间 = 100 × 100 × 8 × 3 = 240,000。

## 训练 2000 轮流程

每 episode 的循环：**选动作 → 执行 → 得到 `(s', r, done)` → 更新 Q 表 → 经验回放**

```
for ep in range(2000):
    state = env.reset()
    for step in range(500):
        action = 选择动作(state)
        next_state, reward, done = env.step(action)
        更新 Q 表(state, action, reward, next_state)
        if done: break
    经验回放()
    epsilon 衰减()   # 仅 Alg2
```

### 二者区别：动作选择策略

| | Alg2 (QL+ε-greedy) | Alg4 (SDP-QL) |
|---|---|---|
| 策略 | ε 概率随机选；否则选 Q 值最大 | SDP 贪心：计算 3 个动作到目标的曼哈顿距离，按距离升序依次尝试第一个能无碰撞执行的动作 |
| 引导 | 纯靠 Q 值 + 随机探索 | 启发式（贪心走近目标）+ Q 值更新 |

## 碰撞检测

USV 有物理尺寸（25m×8m），与障碍物使用 OBB 相交检测（SAT 分离轴定理）。

扫掠碰撞检测：对转弯弧线采样 12 点、对直线前进采样 15 点，逐点用 OBB（附加 10m 安全裕度）与所有可见障碍物做碰撞检测。任意一点碰撞则整条轨迹判为不可行。

## 关于路径最优性

- **Alg2** 输出的是 Q 值最高的路径，受离散网格近似和 Q-learning 收敛精度限制
- **Alg4** 输出的是 SDP 贪心路径，曼哈顿距离本身不是真实最短路径度量，且贪心不考虑前瞻

两者都是**近似可行路径**，不保证全局最短或最优。
