## 1. Q-Learning 更新规则（Q 值迭代）

### 状态-动作空间

$$\text{状态 } s = (g_x, g_y, h) \in \{0,\dots,99\}^2 \times \{0,\dots,7\}$$
$$\text{动作 } a \in \{0(\text{FORWARD}), 1(\text{TURN\_LEFT}), 2(\text{TURN\_RIGHT})\}$$
$$\text{Q 表: } Q(s, a) \in \mathbb{R}^{100 \times 100 \times 8 \times 3}$$

### 贝尔曼最优方程（Q-Learning 变体）

$$Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \cdot \underbrace{\left[ r_t + \gamma \cdot \max_{a'} Q(s_{t+1}, a') - Q(s_t, a_t) \right]}_{\text{TD Error } \delta_t}$$

其中：

| 符号 | 含义 | 取值 |
|------|------|------|
| $\alpha$ | 学习率（Learning Rate） | Alg2: $0.4$，Alg4: $0.3$（默认） |
| $\gamma$ | 折扣因子（Discount Factor） | 渐进增长，见第 5 节 |
| $r_t$ | 即时奖励 | 见第 2 节 |
| $s_t$ | 当前状态 | 第 $t$ 步的 $(g_x, g_y, h)$ |
| $s_{t+1}$ | 下一状态 | 执行 $a_t$ 后的 $(g_x', g_y', h')$ |

**代码**（`update_q`，第 606-611 行）：

```python
td_target = reward + gamma * np.max(q_table[ngx, ngy, nh, :])
q_table[gx, gy, h, action] += alpha * (td_target - q_table[gx, gy, h, action])
```

## 2. 奖励函数

### Alg2 的奖励（含距离塑形）

$$r_t = r_t^{\text{env}} + \Delta d_t \cdot \frac{\lambda_d}{W} - \lambda_{\text{step}}$$

$$\Delta d_t = d_{t-1}^{\text{usv}} - d_t^{\text{usv}} = \sqrt{(x_{t-1} - g_x)^2 + (y_{t-1} - g_y)^2} - \sqrt{(x_t - g_x)^2 + (y_t - g_y)^2}$$

| 符号 | 含义 | 取值 |
|------|------|------|
| $r_t^{\text{env}}$ | 环境奖励 | $-1$（碰撞），$0$（正常），$+1$（到达目标） |
| $\lambda_d$ | 距离奖励系数 | $0.05$ |
| $W$ | 世界尺寸归一化 | $1000$ |
| $\lambda_{\text{step}}$ | 每步惩罚 | $0.001$ |

### Alg4 的奖励（仅环境奖励 + 步惩罚）

$$r_t = r_t^{\text{env}} - \lambda_{\text{step}}$$

Alg4 无距离塑形项，其探索完全由 SDP 启发式引导。

### 奖励取值示例表

| 场景 | $r_t^{\text{env}}$ | 距离项 | $r_t$ |
|------|-------------------|--------|-------|
| 撞墙 / 撞障碍物 | $-1$ | $0$（跳过计算） | $-1.001$ |
| 正常移动，靠近目标 10m | $0$ | $+10 \times 0.05 / 1000 = +0.0005$ | $-0.0005$ |
| 正常移动，远离目标 10m | $0$ | $-10 \times 0.05 / 1000 = -0.0005$ | $-0.0015$ |
| 到达目标 | $+1$ | 有距离项 | $\approx +0.999$ |

## 3. ε-greedy 动作选择

以概率 $\varepsilon$ 随机探索，以概率 $1 - \varepsilon$ 选择当前 Q 值最大的动作：

$$a_t = \begin{cases}
\text{Uniform}(0, 1, 2) & \text{with probability } \varepsilon \\
\arg\max_{a} Q(s_t, a) & \text{with probability } 1 - \varepsilon
\end{cases}$$

若有多个动作同为最大值，则从中随机等概率选择：

$$a_t \sim \text{Uniform}\Big( \{ a \mid Q(s_t, a) = \max_{a'} Q(s_t, a') \} \Big)$$

**代码**（第 553-561 行）：

```python
if random() < epsilon:
    action = randint(0, N_ACTIONS)       # ε: 随机探索
else:
    q_vals = q_table[gx, gy, h, :]
    max_q = max(q_vals)
    best = [a for a, v in enumerate(q_vals) if v == max_q]
    action = choice(best)                # 1-ε: 最优利用
```

## 4. Epsilon 衰减

每 episode 结束后按指数衰减：

$$\varepsilon_{k+1} = \max(\varepsilon_{\min},\; \varepsilon_k \cdot \eta)$$

| 符号 | 含义 | 取值 |
|------|------|------|
| $\varepsilon_0$ | 初始探索率 | $1.0$ |
| $\eta$ | 衰减因子 | $0.995$ |
| $\varepsilon_{\min}$ | 探索率下限 | $0.02$ |
| $k$ | Episode 编号 | $1, 2, \dots, 2000$ |

闭合形式：

$$\varepsilon_k = \max(0.02,\; 1.0 \times 0.995^k)$$

到第 $\approx 780$ 集触底，此后稳定在 2% 探索。

**代码**（第 547-548 行）：

```python
def decay_epsilon(self):
    self.epsilon = max(self.epsilon_min, self.epsilon * self.epsilon_decay)
```

## 5. Gamma（折扣因子）渐进调度

$$\gamma(k) = \gamma_{\text{start}} + (\gamma_{\text{end}} - \gamma_{\text{start}}) \cdot \frac{k}{K}$$

| 符号 | 含义 | 取值 |
|------|------|------|
| $\gamma_{\text{start}}$ | 初始折扣因子 | $0.1$ |
| $\gamma_{\text{end}}$ | 最终折扣因子 | $0.9$ |
| $K$ | 总 Episode 数 | $2000$ |
| $k$ | 当前 Episode 编号 | $0, 1, \dots, K$ |

设计意图：训练早期 $\gamma$ 低，鼓励关注短期奖励（快速到达目标）；训练后期 $\gamma$ 高，鼓励长视距规划（考虑远距离目标回报）。

**代码**（第 540-542 行）：

```python
def get_gamma(self, episode):
    progress = min(episode / self.episodes, 1.0)
    return gamma_start + (gamma_end - gamma_start) * progress
```

## 6. 经验回放（Experience Replay）

从经验池 $\mathcal{D}$ 中随机抽取小批量样本进行额外的 Q 值更新：

$$(s_j, a_j, r_j, s'_j) \sim \text{Uniform}(\mathcal{D}), \quad j = 1, \dots, B$$

$$Q(s_j, a_j) \leftarrow Q(s_j, a_j) + \alpha \cdot \left[ r_j + \gamma \cdot \max_{a'} Q(s'_j, a') - Q(s_j, a_j) \right]$$

| 符号 | 含义 | 取值 |
|------|------|------|
| $\mathcal{D}$ | 经验池（Replay Buffer） | 最大容量 $5000$，FIFO |
| $B$ | 批量大小（Batch Size） | $32$ |

每个 episode 结束时执行一次（先在线更新 Q，再批量回放）。

**代码**（第 613-622 行）：

```python
indices = choice(len(replay_buffer), batch_size, replace=False)
for idx in indices:
    s, a, r, ns = replay_buffer[idx]
    td_target = r + gamma * max(q_table[ns])
    q_table[s] += alpha * (td_target - q_table[s])
```

## 7. SDP（Shortest Distance Path）启发式

### 曼哈顿距离

$$d_{\text{manhattan}}(p, g) = |p_x - g_x| + |p_y - g_y|$$

### 动作排序

对每个动作 $a$，模拟执行后得到假设位置 $(x_a, y_a, h_a)$，计算：

$$d_a = d_{\text{manhattan}}\big((x_a, y_a),\; g_{\text{goal}}\big)$$

按 $d_a$ 升序排列，$d_a$ 越小优先级越高。碰撞不可行的动作被排除。

$$\text{Candidates} = \text{SortAscending}\Big( \{ (a, d_a) \mid a \text{ 可无碰撞执行} \} \Big)$$

**代码**（第 563-579 行）：

```python
def rank_actions_sdp(state):
    candidates = []
    for a in range(N_ACTIONS):
        result = temp_usv.try_action(a, visible, occ)
        if result is not None:
            nx, ny = result[0], result[1]
            d = abs(nx - goal[0]) + abs(ny - goal[1])
            candidates.append((a, d))
    candidates.sort(key=lambda x: x[1])
    return candidates
```

### SDP 纯粹策略（无 Q 表，仅用于评估）

$$a_t^{\text{SDP}} = \arg\min_{a \in \text{feasible}} d_{\text{manhattan}}\big(\text{result}(s_t, a),\; g_{\text{goal}}\big)$$

**代码**（第 581-604 行 `choose_action_sdp`）。

## 8. Alg4 SDP-QL 训练流程

SDP 引导 + Q 值更新的组合训练。每步流程：

**Step 1**：SDP 排序动作

$$(a_1, d_1), (a_2, d_2), \dots \leftarrow \text{rank\_actions\_sdp}(s_t)$$

**Step 2**：依次尝试候选，取首个成功的

遍历 $(a_i, d_i)$，执行 $\text{env.step}(a_i)$：
- 若 $r_t^{\text{env}} < 0$（碰撞）→ 继续下一个候选
- 若 $r_t^{\text{env}} \geq 0$（可行）→ 接受该动作

**Step 3**：Q 值更新

$$Q(s_t, a_i) \leftarrow Q(s_t, a_i) + \alpha \cdot \left[ r_t + \gamma \cdot \max_{a'} Q(s_{t+1}, a') - Q(s_t, a_i) \right]$$

**Step 4**：若所有候选均碰撞 → Episode 终止。

**代码**（第 825-862 行）：

```python
candidates = rank_actions_sdp(state)
for a, d in candidates:
    next_state, reward, done = env.step(a)
    if reward < 0:
        continue        # SDP 第一候选失败 → 重规划, 换下一候选
    else:
        update_q(state, a, reward, next_state, gamma)
        state = next_state
        break
```

关键特征：SDP 提供"好动作"的优先顺序，Q-learning 提供"动作价值"的长期学习，二者互补。

## 9. Alg2 QL+ε-greedy 训练流程

纯粹依靠 ε-greedy 探索学习。每步流程：

**Step 1**：ε-greedy 选动作

## 10. 两种算法核心对比

| 维度 | Alg2（QL+ε-greedy） | Alg4（SDP-QL） |
|------|---------------------|----------------|
| 探索策略 | ε-greedy（随机探索） | SDP 排序（贪心引导） |
| 动作选择 | $\arg\max Q(s,a)$ 或随机 | 按 $d_{\text{manhattan}}$ 升序依次尝试 |
| 奖励塑形 | 有（距离变化奖励） | 无（仅环境奖励） |
| 依赖启发式 | 否（纯强化学习） | 是（SDP 曼哈顿距离） |
| 收敛速度 | 慢（需大量试错） | 快（启发式直接引导） |
| 最优性保证 | 无（表格 Q-Learning 近似） | 无（贪心非全局最优） |

## 11. Q 表维度汇总

```
Q ∈ R^{100 × 100 × 8 × 3}        = 240,000 个参数
        ↓     ↓   ↓  ↓
     grid_x grid_y hdg act

grid_x, grid_y : USV 连续坐标离散化到 10m 网格 (0~99)
hdg            : 8 个航向角 {0°, 45°, 90°, 135°, 180°, 225°, 270°, 315°}
act            : 3 个动作 {FORWARD, TURN_LEFT, TURN_RIGHT}
```

虽仅有 3 个离散动作，但物理执行时：
- **转弯**使用连续弧线（8 个转弯半径档位：15m → 60m），同一航向下有 8 种轨迹
- **直线前进**使用连续直线（采样 15 点做扫掠碰撞检测）
- 因此实际可达轨迹为 $3 \times 8 \times 8 = 192$ 种物理路径组合

## 12. DQN（Deep Q-Network）+ SDP

### 核心思想

用神经网络 $Q_\theta(s, a)$ 替代 Q 表，参数 $\theta$ 通过梯度下降最小化 TD 误差。

与 SDP 的结合方式同 Alg4：SDP 按曼哈顿距离排序动作，依次尝试；DQN 负责 Q 值函数逼近。

### 状态编码

DQN 输入不是离散索引，而是归一化的连续向量：

$$\mathbf{x}(s) = \big( \frac{g_x}{99},\; \frac{g_y}{99},\; \underbrace{0,\dots,1,\dots,0}_{\text{8-dim one-hot heading}} \big) \in \mathbb{R}^{10}$$

- 网格坐标归一化到 $[0, 1]$（保持平移不变性）
- 航向角用 8 维 one-hot 编码

### 网络结构

三层全连接 MLP，隐藏层用 ReLU 激活，输出层线性：

$$\begin{aligned}
\mathbf{z}^{(1)} &= \mathbf{x} W_1^\top + \mathbf{b}_1, & W_1 &\in \mathbb{R}^{64 \times 10} \\
\mathbf{h}^{(1)} &= \text{ReLU}(\mathbf{z}^{(1)}) = \max(0, \mathbf{z}^{(1)}) \\
\mathbf{z}^{(2)} &= \mathbf{h}^{(1)} W_2^\top + \mathbf{b}_2, & W_2 &\in \mathbb{R}^{64 \times 64} \\
\mathbf{h}^{(2)} &= \text{ReLU}(\mathbf{z}^{(2)}) \\
Q_\theta(s, \cdot) &= \mathbf{h}^{(2)} W_3^\top + \mathbf{b}_3, & W_3 &\in \mathbb{R}^{3 \times 64}
\end{aligned}$$

权重初始化：Xavier/Glorot Uniform

$$W_{ij} \sim \text{Uniform}\left(-\sqrt{\frac{6}{n_{\text{in}} + n_{\text{out}}}},\; \sqrt{\frac{6}{n_{\text{in}} + n_{\text{out}}}}\right)$$

### TD 目标与损失函数

DQN 使用 MSE 损失 + 目标网络：

$$\mathcal{L}(\theta) = \mathbb{E}_{(s, a, r, s') \sim \mathcal{D}} \left[ \left( Q_\theta(s, a) - y \right)^2 \right]$$

$$\text{TD目标: } y = r + \gamma \cdot \max_{a'} Q_{\theta^-}(s', a')$$

其中 $\theta^-$ 为目标网络（Target Network）参数，是 $\theta$ 的延迟副本。

**仅对被选中的动作 $a$ 的 Q 值计算损失**，构建目标向量：

$$y_{\text{batch}}[i] = \begin{cases}
r_i + \gamma \cdot \max_{a'} Q_{\theta^-}(s_i', a') & \text{对于被选动作 } a_i \\
Q_\theta(s_i, a) & \text{对于未被选的动作}
\end{cases}$$

**代码**（`_train_network_step`，第 967-990 行）：

```python
q_online = online_net.forward(x_batch)           # (batch, 3)
q_target_next = target_net.forward(x_next_batch) # (batch, 3)
max_next_q = max(q_target_next, axis=1)          # (batch,)
targets = batch_r + gamma * max_next_q           # TD target

y_batch = q_online.copy()
for i, a in enumerate(batch_actions):
    y_batch[i, a] = targets[i]  # 仅更新被选动作的 Q 值

loss = online_net.backward(x_batch, y_batch)
```

### 目标网络更新（硬更新）

每 $C = 10$ 个 episode，将在线网络权重完整复制到目标网络：

$$\theta^- \leftarrow \theta$$

$$W_i^- := W_i,\quad \mathbf{b}_i^- := \mathbf{b}_i$$

### 梯度下降（前向 / 反向传播）

**前向**（逐层）：

$$\underbrace{\mathbf{x}}_{\mathbb{R}^{B \times 10}} \xrightarrow{W_1, \mathbf{b}_1} \mathbf{z}^{(1)} \xrightarrow{\text{ReLU}} \mathbf{h}^{(1)} \xrightarrow{W_2, \mathbf{b}_2} \mathbf{z}^{(2)} \xrightarrow{\text{ReLU}} \mathbf{h}^{(2)} \xrightarrow{W_3, \mathbf{b}_3} Q \in \mathbb{R}^{B \times 3}$$

**反向**（MSE 损失梯度 + 链式法则）：

$$\begin{aligned}
\frac{\partial \mathcal{L}}{\partial Q} &= \frac{1}{B}(Q - Y) \in \mathbb{R}^{B \times 3} \\[4pt]
\frac{\partial \mathcal{L}}{\partial W_3} &= \left(\frac{\partial \mathcal{L}}{\partial Q}\right)^\top \mathbf{h}^{(2)} \in \mathbb{R}^{3 \times 64} \\
\frac{\partial \mathcal{L}}{\partial \mathbf{b}_3} &= \sum_{i=1}^B \frac{\partial \mathcal{L}}{\partial Q_i} \in \mathbb{R}^{3} \\[4pt]
\frac{\partial \mathcal{L}}{\partial \mathbf{h}^{(2)}} &= \frac{\partial \mathcal{L}}{\partial Q} W_3 \in \mathbb{R}^{B \times 64} \\
\frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(2)}} &= \frac{\partial \mathcal{L}}{\partial \mathbf{h}^{(2)}} \odot \mathbb{1}[\mathbf{z}^{(2)} > 0] \quad (\text{ReLU 导数}) \\[4pt]
\frac{\partial \mathcal{L}}{\partial W_2} &= \left(\frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(2)}}\right)^\top \mathbf{h}^{(1)} \\
\frac{\partial \mathcal{L}}{\partial \mathbf{b}_2} &= \sum_{i=1}^B \frac{\partial \mathcal{L}}{\partial \mathbf{z}_i^{(2)}} \\[4pt]
&\vdots \\[4pt]
\frac{\partial \mathcal{L}}{\partial W_1} &= \left(\frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(1)}}\right)^\top \mathbf{x}
\end{aligned}$$

**权重更新**（SGD）：

$$W_i \leftarrow W_i - \eta \cdot \frac{\partial \mathcal{L}}{\partial W_i}, \quad \mathbf{b}_i \leftarrow \mathbf{b}_i - \eta \cdot \frac{\partial \mathcal{L}}{\partial \mathbf{b}_i}$$

其中 $\eta = 0.001$（学习率）。

### 经验回放（DQN 专用）

缓冲池 $\mathcal{D}_{\text{DQN}}$ 容量 $10000$，FIFO。

每个 episode 结束时从中采样 $B = 64$ 条经验做一次批量梯度下降。

每次存入经验时不立即训练（与 Q 表版本不同），而是**每 episode 结束后统一训练**。

### DQN+SDP 训练流程（Alg5）

```
for ep in range(2000):
    state = env.reset()
    for step in range(500):
        candidates = SDP_rank(state)       # SDP 排序动作
        for a in candidates:               # 依次尝试
            s', r, done = env.step(a)
            if r < 0: continue             # 碰撞 → 换下一候选
            replay_buffer.append((s, a, r, s'))
            s = s'
            break
    训练一次 DQN (mini-batch)
    if (ep+1) % 10 == 0:
        target_net ← online_net            # 目标网络更新
```
